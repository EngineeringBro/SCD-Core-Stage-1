RepairQ utilizes QZ Tray integration for seamless printing without navigating print dialogs. This provides users with a one-click printing process for receipts, invoices, and labels directly to their assigned printer. Users can also create and customize print automation for a hands-free process for printing, further streamlining your ticket workflow.

Once you have paired your printer(s) with your computer, installed QZ, and turned on Integrated Printing in RepairQ, you can begin adding printers to the integration. Once you have completed this process, please see our additional resources for further integration setup and troubleshooting.

Add an Integrated Printer to RepairQ

1. On the Navigation Bar, go to Settings → Location Settings.

2. On the Settings Overview page, there is a list of Settings Categories in the left column. Click Integrated Printing to expand the options and select Printers.

3. On the Printer Overview page, click Add Printer on the right side of the page.

4. Fill out the printer settings and make sure all required fields are populated with a selection. Below is a list of options when adding a printer. Any in bold are required/recommended settings. Once you are finished entering the print settings, click Save to finalize your changes.

Note: The available settings may vary based on the type of printer selected. You will need to repeat these steps with any additional printers you plan on using with Integrated Printing.

  • Printer Type - Select the printer type for this setup: Desktop, Receipt, or Label.

  • Printer Name - These names will correspond with printers that you have added to your computer.

  • Location - Select the assigned location for this printer. Admins can choose a specific location or all locations.

  • Description - Add a description of this printer for reference.

  • Color Type - Select a print color: Black & White, Grayscale, or Color

  • Orientation - Portrait, Landscape, or Reverse Landscape for paper width/height.

  • Rotation - Default, No Rotation, 90 degrees clockwise, 180 degrees, or 90 degrees counter-clockwise.

  • Unit of Measurement - Metric (millimeters) or Imperial (inches). Some Printers work best using millimeters. Consult the user manual for your specific printer to determine the exact dimensions. The default is Metric (millimeters) so users have more precision using millimeters.

  • Paper Width/Height - We recommend using millimeters for more exact dimensions for receipts and labels.

  • Image Width/Height - We recommend using millimeters for more exact dimensions for receipts and labels.

  • Scale to Fit - Selecting this will scale the print job to fit the paper size. Try both settings to determine which works best for the printer.

  • Top/Bottom Margin - We recommend starting at zero for each margin and adjusting as needed.

  • Right/Left Margin - We recommend starting at zero for each margin and adjusting as needed.

  • Printer DPI/DPIM - (Dots Per Inch/Millimeter) This controls the resolution of the print. Check your printer specs to determine what DPI is compatible with your device. We recommend starting a 0 and adjusting as needed if the default doesn't work.

  • Rasterize - Converts images into a raster (pixel) image for printing.

Integrated Printing From a Ticket

When viewing a ticket, click the Print Receipt button along the top row of commands.

Note: As a backup, you can print from a browser dialog by clicking the dropdown next to the print icon and selecting Print With Browser Dialog.

 

A Status window will appear with the current status of the print job. Once the job is sent, the window will close automatically. Check your printout to confirm it was sent. If you need to make modifications to your margins, DPI, width, or height, go back to Location Settings → Integrated Printing → Printers and edit the printer settings.
