Released 8/8/2018 10:55 CT (15:55 UTC)

Login & Permissions Optimization

We optimized the login screen and the permissions checking functions. There should be a noticeable load time improvement with the login, and on most list pages.

Popover Updates

Popovers now auto-dismiss more effectively when you move your cursor away from the target.

Dashboard Announcements

We resolved an issue where Admins were unable to see announcements in some cases.

Customer Add/Edit on Tickets

We resolved an issue where the labels and values on the ticket screen would not always line up properly when adding/editing a customer from the ticket.

We also added the company name to the customer view and the ticket view screens.

Customer List Sorting

We fixed an issue that would prevent sorting customers by columns while a name/phone number search was applied.

Pending Shipment Notifications

We fixed an issue that prevented notifications for the "Pending Shipment" shipment status.

Purchase Order Transfers with Markup

We fixed an issue where an unset cost markup percentage would result in a zero price.
