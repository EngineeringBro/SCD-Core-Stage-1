The Inventory Valuation Summary and Detail Reports calculate the total cost and price for a set of inventory items.

We recommend exporting this report on a periodic basis to retain a point-in-time snapshot of your inventory. RepairQ does not keep a historical record of your inventory levels from a previously given date.

Viewing Inventory Valuation Reports

1. On the Navigation Bar, go to Reports.

2. On the Reports page, click on the Inventory dropdown along the top row of report categories. From the available list, click Valuation Summary.

Note: If you would like to see a breakdown of this information, click Inventory Detail.

3. Use the available filters to scope results based on the assigned parameters.

  • Date Range - Select a range of dates for your scope of results.

  • Date Type - Use in combination to find items within the date range.

    • Date Created - Used to report for inventory items created (added into the system).

    • Date Updated - Used to report for inventory items updated in the date range, regardless of when it was added into the system.

  • Location - Select a specific location or run an aggregate report for multiple stores.

  • Status - Use to filter the report for items in one or more inventory statuses. This looks for items that are currently in the selected status. For an explanation of inventory statuses, see Inventory Status - What Does It Mean?

  • Condition - Used to select inventory classified under a specific condition.

  • Supplier - Used to select inventory from a specific supplier.

  • Include Inactive Catalog Items - Select to include inventory items that may still exist in your inventory, but the catalog entry has been deactivated for use.

Note: While consumed inventory statuses (Sold, RMA Credit, Parts Only, Damaged, Shrinkage, Voided) are listed here, we recommend using the Inventory Usage Report for finding the valuation of these items.

4. Once you have selected your filters, click Apply Filters to run the report.

Note: You can click Hide/Show Filters on the upper-right side of the page to collapse/expand the filters when viewing reports. You can also toggle between the summary and detail report from this section.

5. (Optional) Click To Excel on the upper-right side of the report page to export an Excel-friendly document. The export will include any filtered data when running the report.

Inventory Valuation Summary

The Inventory Valuation Summary report will show a summarized list of item types, quantities, and various sums of Cost/Price

Inventory Valuation Detail

The Inventory Valuation Detail report will show a breakdown of catalog items, quantities, and various sums of Cost/Price.

If you would like to view additional historical data for an inventory item, click View Details. You will be taken to the Inventory Summary page where you can see a full list of past and current inventory items with their Inventory Status.

Valuation of Current In-stock Inventory

Below is an example of inventory filters to select to see your current in-stock valuation. This will only pull results for inventory items that are currently assigned to Instock status. This will not include inventory that may still reside at your location but are in transition status (Pulled, Pending Transfer, Pending Refurb, etc.).

  • Date - Leave blank. Click the Cancel (X) button to clear the field.

  • Date Type - Date Created

  • Location - Select a specific location

  • Status - Instock

  • Condition - All

  • Supplier - All

  • Include Inactive Catalog Items - Checked

Finding Old Inventory

Below is an example to run a report at the end of the year to identify stale inventory that may need to be discounted in order to move product.

  • Date Range - Select a date range beginning at the first of the year.

  • Date Type - Date Created

  • Location - Select a location

  • Status - Instock

  • Condition - All

  • Supplier - All

  • Include Inactive Catalog Items - Checked
