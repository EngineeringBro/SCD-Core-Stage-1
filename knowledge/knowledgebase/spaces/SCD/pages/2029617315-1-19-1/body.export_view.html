New Features

  • Catalog Quick Create

    • Quick Create allows users to add new catalog items and onhand quantities without leaving a ticket.

    • Quick Create is also available when creating a new Purchase Order.

    • The catalog can be sorted by Created Item Context, so Managers can see details about when, by whom, and by which process catalog items were created.

Bug Fixes

  • Fixed an issue affecting the Weekly Total column of the Attendance page.

  • Fixed an issue that restricted users outside of the Manager role from selecting the Repair Complete status on tickets.

  • Fixed an issue for some users where Reorder Points set by location override were not triggering items to be reordered.
