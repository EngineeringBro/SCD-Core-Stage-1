Bug Fixes

  • Fixed an issue that caused an ‘Import failed’ message when importing catalog items.

  • Resolved an issue where the location scope for Catalog Items was incorrectly changing from ‘Global’ to ‘Location' after saving changes, causing some items to disappear from view in the catalog.

  • Resolved an issue where the effective tax-inclusive price for Inventory Labels ignored the tax-inclusive app setting, adding tax to an already tax-inclusive price.

  • Fixed an issue where adding a partial cash payment incorrectly set the ticket balance to $0. Now, only the entered payment amount will be applied, ensuring the remaining balance is correctly reflected.

  • Resolved an issue where the ‘Not a Customer Yet’ checkbox was not syncing between the General and Misc tabs when creating a Customer Profile. Now, any change made in one tab will automatically update in the other.

  • Resolved an issue where searching for a customer name containing an apostrophe in the Contacts search field caused a 500 error.

  • Fixed an issue where Supplier SKUs were not appearing in the catalog item view. Catalog items with an assigned Supplier SKU are now visible in the Catalog Item list.
