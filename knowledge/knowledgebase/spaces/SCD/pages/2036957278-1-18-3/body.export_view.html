Released 04/04/2022 at 11:30pm CT

Minor Changes

  • Improved Square Integration so that it requires logging in with the correct credentials, even if a user was previously logged into Square on that browser session.

  • Improved the Square Integration's UI for clarity, based on client feedback.

  • On the Cash Reconciliation page, separated "payment amount" and "tip amount from" in the Gateway Transactions list, for clarity.

Bug Fixes

  • For the Square Integration, resolved an issue where the "Tip First" and "Tip Last" options were not working correctly.

  • Fixed an issue where trying to download a Ticket Approval redirected the user to a failure error page.

  • Resolved behavior on internal purchase orders where items rejected on the order would still create unwanted items in "Backordered" status.

  • For certain customers utilizing GSX service programs, implemented multiple bug fixes for the FedEx integration, especially resolving an issue where the RMA Tracking page being filtered for "sent" items caused an Error500.

  • Resolved a problem where clicking the Import Compatibility download button would not do anything.

  • Fixed a bug where uploading Branding Images worked inconsistently.

  • Fixed a bug where replacing Branding Images worked inconsistently.

  • Fixed a bug where In some cases, Catalog Items would not display under the correct category tree in Qsite.
