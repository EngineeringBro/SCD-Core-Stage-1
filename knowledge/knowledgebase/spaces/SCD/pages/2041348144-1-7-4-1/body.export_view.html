This update includes a few bug fixes for the new features in 1.7.4

Inventory Count Sheet

  • Save & Submit is not submitting the last counted item - the counted item in the "Current Item" section would not always be submitted with the "Save & Submit" action. This has been fixed.

  • UPC scan is pulling in deleted catalog items - fixed.

Inventory Count Reconciliation

  • When all serials are valid, "Save and Mark Reconciled" button was not active. Fixed.

  • Improved the pop-over window sizing

Custom Form Settings

  • Form preview - you could select more than one radio button value. Fixed, now you can only select one value for each radio button in the preview window. NOTE: this issue did not affect actual forms (i.e. forms you fill out and save on tickets)
