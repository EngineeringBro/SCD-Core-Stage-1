Catalog Quick Create allows users to create new catalog items directly from purchase orders and tickets. Creating a catalog item from a ticket will also allow you to update onhand quantities of inventory, so you can add and stock catalog items in real-time, without leaving the current ticket.

There are a few things to remember for Quick Create items:

  • Items are created at the Global level, i.e. all locations will have access to this catalog item.

  • All roles that have access to creating tickets can quick-create catalog items.

  • Admins can choose which Item Types are used for Quick Create.

We recommend checking out the instructions below to ensure quick create features are ready for use, as well as referencing additional resource links for tutorials on ticket and purchase order processes and enhancements added with Quick Create.

Allowing Quick Create for Ticket Items and Purchase Orders

1. On the Navigation Bar go to Settings → Company Settings.

2. On the Settings Overview page, click the Catalog dropdown on the left column of settings. From the dropdown list, click General Settings.

3. On the Ticket General Settings page, there are sections for Ticket Items, Repair Devices, Trade-in Items, and Purchase Orders. Under each section, there is a setting to toggle activation for the feature and additional customization settings. For each active field, you will need to add which Item Types will be available to the user when creating a new item. After you have made your selections, be sure to Save your settings on the upper-right of the page.

  • On/Off Toggle - This will turn on/off the ability to add items from the corresponding ticket/PO.

  • For these item types - Select which Item Types can be used when creating a new Catalog Item.

  • Default Type - Select the default Item Type selection when creating a new item.
