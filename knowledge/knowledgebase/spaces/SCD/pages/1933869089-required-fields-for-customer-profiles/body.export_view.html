Admins can configure which information fields are required when creating or editing a Customer Profile. At a minimum, RepairQ requires a First Name, Last Name, and Primary Phone number to create a profile. Fields marked as required cannot be bypassed when creating a new customer, and any new requirements will also apply when editing existing customer profiles. This ensures that essential customer information is consistently captured, improving data accuracy and compliance with business needs.

Customer Required Fields are available to users on RepairQ Advanced and RepairQ Professional plans.

Updating Required Fields for Creating New Customers

From the Navigation Bar, go to Settings → Company Settings. On the Settings Overview page, under the Customers section, select Required Fields.

On the Validation Rule Settings screen, you can select additional required fields when adding a new customer. By default, RepairQ requires unique entries for First Name, Last Name, and Primary Phone Number. In the image below, the user has added Email and Zip Code to the list of required fields. Once you're satisfied with your selections, click Save to apply the changes.

Required Fields when Creating a Profile

When creating a customer, additional required fields will be marked with an asterisk (*) next to the field. The tab for any sub-category where additional required fields are located will also display an asterisk. For better visibility, required fields will appear on the General tab during the customer creation process.

Note: If you add a new required field, it will apply to new customers and when editing existing customer profiles. Additionally, if Zip Code Validation is enabled, the Zip Code field must be filled out for all customers.
