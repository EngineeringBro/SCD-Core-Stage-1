Released 08/29/2022 at 11:30pm CT

Improvements

  • Users can now enable and disable an Open Item List (found under Company Settings → Tickets → General Settings) which toggles the list of open tickets and appointments a customer has open when adding them to a new ticket.

  • Broadened the ticket statuses that Appointment Reminders can be sent out from to include "Technician Assigned", "Accepted", and "Rejected" statuses.

Bug Fixes

  • Resolved an issue where trying to create any ticket type would sometimes return an error preventing progression.

  • The ticket balance and customer balance, which had been previously calculating incorrectly in automated notifications, now display as intended.

  • Fixed the Appointment Summary "Start" button, which previously did not display any text.

  • Square terminals no longer allow transactions when the integration is turned off.

  • Resolved an issue where RepairQ was calculating a customer's available credit incorrectly after a store credit transaction was removed from a ticket.

  • Return repairs now show custom forms when saved.
