Staff can print receipts for tickets and invoices at any time from the Ticket View page.

Note: Since RepairQ is cloud-based, you will need to set up your printer with your computer, first, in order to print from your browser.

Printing a Receipt or Invoice from a Ticket

1. Search for an existing ticket, or create a new ticket and save it in a new status.

2. When viewing the ticket, available Ticket Actions are along the top row of the ticket.

3. To print, click either the Print Receipt button (printer icon) or the Print Invoice button (page icon). This will open a print dialog preview page where you can select the printer destination.

Integrated Printing

If you have set up Integrated Printing, these icons will look slightly different. Clicking a Print icon will automatically send the job to the printer designated in your Print Config settings.

If you click the dropdown next to the icon, you can choose from the list of printers configured in your Automated Printing settings or print with Browser Dialog.

Note: The Browser Dialog page will open in a new window and allows you to select the destination, layout, and other settings.
