This is a minor release with a few bug fixes.

Summary Report

Tax group sum totals were not always summed correctly. This issue has been resolved.

Miscellaneous App Settings Additions

We added a few new app settings to the "Miscellaneous" page to provide additional configuration controls.

  • Show All Locations on the Leaderboard: When enabled, this setting will allow store managers to see all locations on the location leaderboard report.

  • Allow viewing tickets from any location: When enabled, this adds view-only access for tickets at all locations, even where a user does not have roles assigned.
