Released 05/09/2017 11:21 CT (16:21 UTC)

Sales Summary by Staff Report

Two fixes are submitted in this update.

  • The report now applies the catalog item filter accordingly

  • The sales % column is fixed (i.e. no longer displaying zeros)

Device view, ticket link

The ticket link on the device view is fixed.

Inventory Refurb Permission

The permission rule for creating refurb tickets from inventory items has been separated out from the base inventory edit permission; the permission can now be configured separately from inventory editing (e.g. allowing refurb creation without also allowing inventory editing).

Permissions customization is only available on the Enterprise Plan

Repair Device Notes

The spacing for line/paragraph breaks was "doubled" when the new text markup tool was introduced in RepairQ 1.12.5. This issue has been resolved, and the spacing is now what it was before the 1.12.5 update.
