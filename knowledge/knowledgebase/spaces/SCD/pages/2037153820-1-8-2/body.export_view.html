This is a minor release with a few bug fixes.

Inventory Summary Excel

This was pointed at the wrong excel template (Inventory Item Details Excel), and is now corrected.

Removing Ticket Items from a Return Repair Ticket

Ticket items could not be removed from a return repair ticket due to an error. This is now fixed.

Batch Invoice Print

The print template for batch invoices was not loading due to an error. This has been fixed.

Account Management

The minimum users and locations required when updating your RepairQ license was not including the user and location that come with the base license. This has been resolved (effectively reducing the minimum for each by 1)
