Viewing the Appointment List

On the Navigation Bar, go to Opportunities → Appointments.

Navigating List Views

Scheduled appointments can be viewed by List, Day, Week, or Month. When viewing each style, you can cycle between views with the left and right arrows. There is also a Today button that appears when viewing past or future appointments in either Day, Week, or Month, which will bring you back to the current date.

  • List View (Displayed in months)

  • Day View

  • Week View

  • Month View

Viewing an Appointment

Click on an available appointment to view details, set an assignee, make edits to the appointment, or start the appointment.
