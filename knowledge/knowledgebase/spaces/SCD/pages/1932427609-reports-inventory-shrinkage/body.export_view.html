The Inventory Usage Summary Report is an easy way to view all inventory that has been adjusted out as Shrinkage for any given amount of time.

Viewing Shrinkage History with the Inventory Usage Report

1. On the Navigation Bar, go to Reports.

2. On the Reports page, click on the Inventory dropdown along the top row of report categories. From the available list, click Inventory Usage Summary.

Note: You can view a detailed list of inventory items that are in Shrinkage status by selecting Inventory Usage Detail.

3. Use the available filters to scope results based on the assigned parameters. Once you have set your filters, click Apply Filters to run the report.

  • Date Range - Select a date range for your scope of results.

  • Location - Select a specific location or run an aggregate report for multiple stores.

  • Supplier - This is used to filter your search for items from a specific supplier.

Note: The date filters will search for items based on the date an item was moved into a consumed inventory status. If an inventory item is attached to a ticket, then the effective ticket date (the date the ticket status was last updated) will be used for the date filter so that you can match the inventory usage reports up with the other sales and accounting reports in RepairQ.

Note: You can click Hide/Show Filters on the upper-right side of the page to collapse/expand the filters when viewing reports. You can also toggle between the summary and detail report from this section.

4. (Optional) Click To Excel on the upper-right side of the report page to export an Excel-friendly document. The export will include any filtered data when running the report.

Shrinkage (Usage) Summary

The Inventory Usage Summary report will show a summarized breakdown of item types, quantities, and various cost/price details for each consumed status. You can click on each status tab for a breakdown of each category. Click the Shrinkage Tab to see all inventory items that have been adjusted out as shrink.

Shrinkage (Usage) Detail

The Inventory Usage Detail report will show a breakdown of catalog items, quantities, and cost/price details for each consumed status. You can click on each status tab for a breakdown of each category.

The Detail Report also has Item Type and Catalog Item filters. Use these filters to filter the report for specific Item Types or Catalog Item searches.

If you would like to view additional historical data for an inventory item, click View Details. You will be taken to the Inventory Summary page where you can see a full list of past and current inventory items with their Inventory Status.
