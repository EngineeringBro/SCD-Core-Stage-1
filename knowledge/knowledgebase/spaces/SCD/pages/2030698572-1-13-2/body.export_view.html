Released 08/17/2017 3:02 CT (16:02 UTC)

Duplicate Submits

We added protection against importing the same file twice, creating an invoice twice, and saving an RMA batch twice

Ticket View - Return Button

Fixed logic that would hide the return button on the ticket view too aggressively.

Store Credit Updates

In some cases you could apply the same store credit transaction to the customers running credit balance. This has been resolved.

Device Notes Paragraph Margins

We have reduced the margins between paragraphs/line breaks in device notes to restore the margins that existed before the text editor was updated.

Text Editor Spellcheck

Browser spellcheck is now no longer disabled inside the text editor fields (notes, diagnostics, etc)
