1.15.8.1 Bug Fixes (12/18/2018)

  • Versioning fixes to force cache refresh on files

1.15.8.2 Bug Fixes and Enhancements (1/8/2019)

  • Device description is now searchable on the device list

  • Fixed issue with retaining role selection when saving new staff members

  • Ticket browse permission applied to customer and billing agent ticket lists

  • Qpop pairing fix related to timezone handling

1.15.8.3 Bug Fixes and Enhancements (2/7/2019)

  • Catalog item import button is now available when catalog is empty

  • PO notification supplier email tag for internal supplier now uses supplier location's email

  • Fixed issue where ordered items could get orphaned when a backordered item is not fulfilled on the expected purchase order (the item reverts to back ordered when not fulfilled)

  • Location override bin number is now searchable

  • Billing agents no longer default to deactivated for some user roles

  • Updated supplier integration prompts

  • Fixed issue where assignments could not be made to the admin user

  • Fixed default location when purchase order is saved

  • Fixed device list filter reset button

  • Fixed issue with selecting bundle option when adding a new item type

  • Fixed an issue with bundled item price display on ticket invoice template

1.15.8.4 Bug Fixes and Enhancements (2/12/2019)

  • Added the ability to display ticket item SKU barcodes on the ticket receipt and invoice print templates. See print template settings to enable this new feature. 

  • Fixed some issues with processing payments with the SREDKey device and PAX device interchangeably in the same VCT session (before the fix you would need to de-assign and re-assign the VCT with the desired payment device selected).

  • Fixed an issue with picking PAX payment devices on a VCT when both Bluefin and Heartland integrations are enabled at the same time.

  • Improvements to ticket barcode legibility. Barcode scanners should now scan much more effectively.

1.15.8.5 Enhancements (2/13/2019)

  • Refined the ticket barcode width for clearer and more compact printing

1.15.8.6 Bug Fixes and Enhancements (2/20/2019)

  • Dashboard ticket number counts were including quotes, but quotes are not displayed on the ticket queue by default. Quotes are now excluded from the counts.

  • RMA Tracking supplier filter was not as explicit as it should be, and now it will filter for only the selected supplier

  • We added a "select all" to the location filter on the invoice creation screen

  • We fixed an issue with filtering warranty/disclaimers by ticket type

  • When searching inventory for a serial number, the result returned will now be the most recent item, rather than the first record of the serial number
