Released 10/03/16 at 1PM CT

This is a minor release with a few bug fixes.

Return Repairs with Serialized Items

When attempting to replace a serialized item on a return repair ticket, the ticket save would fail due to the error "Failed to create new inventory item for return." This issue has been fixed.

Staff Pick Lists

Some staff pick-lists/select-boxes were not sorted alphabetically by name. Pick lists are now sorted alphabetically.

Optimize Loading Customer Devices

Customers with thousands of devices would load significantly slower than customers with only a few devices. The affected the customer view screen, tickets that the customer was associated with, and the customer search on tickets. We have optimized the loading of customer devices, and now these customers will load as quickly as any other customer.

Ticket Import - Billing Agent ID

The Ticket Import feature now accepts the Billing Agent ID as a matching value.
