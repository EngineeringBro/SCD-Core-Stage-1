The Purchase Order (PO) List View page provides a list of orders created within the last 90 days. If you are searching for older POs or other specific details, the queue can be filtered by status, date created, and other fields. From the List View page, you can also create new POs, view and edit POs to reconcile them, and import POs.

Viewing and Navigating the Purchase Order List View Page

On the Navigation Bar, go to Inventory → Purchase Orders.

The Purchase Order List View is sorted by PO identification (ID) number on the left column. Columns can also be sorted by clicking on the header of the column you would like to sort.

You can use filters to see POs by Supplier, Location, Status, and Keyword Search by ID number.

Note: After applying your filters, click the Apply button to see the results. To clear filters, click the Reset button.

You can save filters as default views and customize what columns of information are displayed by clicking the Options (⋮) button near the top right of the page.

You can start a new Purchase Order by clicking the Add (+) button near the top right of the page, next to the Options (⋮) button.

You can view available actions for a purchase order next to the line item. Under the Actions column, click the Options (⋮) button next to the purchase order for options to View, Edit, or Print the PO.

You can import a PO by clicking the Options (⋮) button and selecting Import. A new import field will appear below your Purchase Order list.

Note: Imported Purchase Orders must be imported as a tab-delimited text file. To download a template with proper formatting for uploading, click Purchase Order Import Template. For more help importing data into RepairQ, check out Importing Purchase Orders.
