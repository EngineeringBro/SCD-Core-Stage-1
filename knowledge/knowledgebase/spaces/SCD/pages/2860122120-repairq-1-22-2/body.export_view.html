System Performance

  • We've resolved an issue that occasionally caused CPU spikes – causing slowdown and longer loading times for some customers – by removing outdated code that was no longer needed. This fix improves system performance and stability.

Catalog Items

  • Resolved an issue with catalog search functionality when the "Enable legacy catalog item search" setting was disabled. This feature now works as expected, allowing users to perform catalog searches without encountering errors.

  • Resolved an issue where clicking the Translations button for a Catalog Item resulted in an Error 500. Now, users can access the Translation Edit page for catalog items as expected when Multilingual settings are enabled.

Payment Processing

  • We've resolved an issue where tipping settings in Square could not be disabled. The issue has been addressed, and now the tipping settings can be activated/disabled as needed.

Import/Export Queue

  • Resolved an issue where the error “Invalid Location” would display when a user imports a PO and selects the option “Read from file.”

  • Resolved an issue where, when the Queued Import feature was active, imports and exports would remain stuck in “Processing” status without completing.
