Released 07/05/2022 at 11:30pm CT

Improvements

  • Improved how Date filters sort within all list view pages.

Bug Fixes

  • Worked through further Square refund issues, where after a pro rata refund would reflect correctly in Square but not in RepairQ.

  • A problem preventing users from counting serialized inventory correctly has been fixed.

  • Internal orders now correctly recalculate open orders when rejected.

  • Disabling Square for one location no longer affects Square for all other locations.

  • Using the Supplier Sku column when exporting inventory no longer causes the exported file to come up blank.

  • The Devices List View now filters results correctly based on filters selected and applied by the user.

  • Resolved an error when users attempted to view Device History on a ticket.
