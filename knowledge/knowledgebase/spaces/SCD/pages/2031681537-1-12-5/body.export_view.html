Released for 05/02/2017 11:18 CT  (16:18 UTC)

New text markup tool

RepairQ is getting a new text markup tool! All of the text markup tools have been replaced with the new version. The new tool is more reliable, is easier to copy & paste into, and comes with a few new formatting options. 

Qpop Pairing Updates

Qpop pairing is now even more reliable. Before this update you could break a paired connection by changing the Qpop companion name in your RepairQ settings, but now this will no longer cause any problems.

Qpop Launch Optimization

Qpop assets have been optimized for faster launch load times.

PAX Payment Device Updates

We fixed an edge-case related to a scenario when multiple card declines occur - in this scenario a sales rep could send a request to the device that would cancel in RepairQ, but would not automatically cancel the request on the PAX device. This could result in a payment request that is processed on the PAX device, but never returned to RepairQ for saving on the ticket.
