Released 3/1/2021 at 11:30pm CT

Fixes and Improvements

  • Blank Catalog Import Template - You can now use the Blank Catalog Import Template button to download a catalog import template with only the column headers and with no catalog data in the template.

  • Column Sort - Some users reported that clicking on certain column headers on the Inventory Count page did not sort the columns as expected. Now, when you click on a column header on that page, the columns will sort as expected.

  • Catalog Export - We fixed an issue where exporting a list of catalog items populated various fields in the export file with incorrect data. This caused a secondary problem where a user making changes to exported data, and then importing that data file back into RepairQ could make major unintentional changes to the catalog.
