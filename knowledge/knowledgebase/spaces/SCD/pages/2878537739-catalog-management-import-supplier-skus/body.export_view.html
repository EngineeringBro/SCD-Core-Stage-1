RepairQ offers convenient inventory management by allowing users to link Supplier SKUs to existing catalog items through the import process. Supplier SKUs help you accurately track and manage inventory, especially when dealing with multiple suppliers or similar products with slight variations. By using the provided Import Template, you can quickly update your inventory records and ensure accurate tracking across your system.

  • Supplier Compatibility - Many suppliers use SKUs to manage their own inventory. Importing these SKUS into your catalog ensures seamless integration with supplier documents, such as invoices and purchase orders.

  • Streamline Reordering - With Supplier SKUs linked to catalog items, creating purchase orders becomes faster and more accurate. This eliminates guesswork when reordering from suppliers.

  • Time-Saving Updates - Bulk importing SKUs allows you to update or sync your catalog quickly, saving time compared to manual entry.

Import capabilities are restricted to users with Admin, Manager, and Inventory Manager roles. This ensures that only authorized personnel handle critical data operations, maintaining data integrity and security.

Download the Supplier SKU Import Template

On the Navigation Bar, go to Settings → Catalog. When viewing the Catalog List, click the Options button near the top right of the page. This will expand the list of available actions. From the list, click Import.

An Import section will appear under the Catalog Item List. Under the “Import Supplier SKUs” header, click Download Supplier SKUs.

This file contains a list of your current Catalog Items with their assigned Supplier SKUs. You can use the existing data to:

  • Edit Assigned Supplier SKUs - Make changes to the SKUs already linked to catalog items.

  • Add New Supplier SKU Information - Clear the data (leaving the headers intact) and enter new information to link catalog items with different or additional Supplier SKUs.

After downloading the template, open the file and add items to the document. Be sure to Save your chagnes before closing the document.

The following columns require data to link Supplier SKU information:

  • Supplier Name (Required) - The name of the Supplier providing the item.

    • This must match an existing supplier in your Supplier List (Settings → Catalog → Suppliers).

  • Catalog Item (Required) - The item in your Catalog that corresponds to the supplier's product.

    • This must match an existing Catalog Item.

  • Supplier SKU (Required) - Enter the specific SKU assigned by the supplier for the product.

  • Vendor Product ID (Optional) - This is optional but can be included if your Supplier provides an additional ID for tracking.

Multiple Supplier SKUs

Catalog items can have multiple Supplier SKUs if the same item is purchased from different suppliers. This allows you to consolidate inventory under a single catalog item while accommodating variations from multiple sources.

Product Quality Tiers

To distinguish between quality (e.g., Name Brand vs. Generic Brand), consider creating separate catalog items for each quality level. This helps maintain clarity in your inventory and sales reporting.

Import the Supplier SKU Template

When you have saved the file and are ready to upload your import, click Choose File and select your file from the file explorer. Once the file is selected, click Import. RepairQ will begin processing your import file.

Import Feedback

After importing a file, you will receive a notification confirming that the import action has been registered, and the file is now queued for processing. You can safely navigate away from the page, as the import will continue to process in the background. Once the import is completed, the user who submitted the import will receive an email. This email includes a link to the page where the import was submitted, as well as a link to our Support page in case any issues arise.

View Assigned Supplier SKUs

To view Supplier SKU details for a Catalog Item, click Supplier SKUs at the top of the item details page.

Note: While editing a Catalog Item, you can also add new Supplier SKUs by clicking the same button.
