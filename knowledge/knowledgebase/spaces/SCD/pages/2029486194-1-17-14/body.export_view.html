1.17.14 is a minor update including some small features and bug fixes.

Scheduled 11:30 CT on October 26, 2020

Improvements

We’ve upgraded the following list views in RepairQ, allowing users to customize which columns appear on these pages and save custom views for you or your team to use.

  • Inventory Transfers

  • RMA Batches

  • Invoices


List View Exports - In a List View, you can add or remove columns. However, until now, when you exported a list, the export file only contained a pre-defined set of columns. The exported list did not reflect the view you created on screen. Now, when you export a list, the export file will contain the columns that you set on screen. This feature is useful for exporting lists/data related to catalog, tickets, inventory and allows you to use List Views for reporting.
