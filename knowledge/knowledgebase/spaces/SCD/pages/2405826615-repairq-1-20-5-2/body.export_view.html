Bugs

Tickets and Transactions

  • Applying a Store Credit on a ticket no longer results in an error.

  • Resolved the issue causing broken images upon uploading a file to a Custom Form.

Inventory and Catalog

  • Fixed a bug that caused a loading screen freeze when adding compatible parts and services to catalog items.

  • Rejecting an item on an Internal Purchase Order will no longer zero out the “Quantity to ship” fields of the page.

Improvements

  • Concurrent appointment fields have been renamed “Maximum Appts” for clarity and consistent behavior across all available settings.

  • Test Mode batch delete errors have been fixed, and any deletion will be recorded in the System Log.

  • Once Test Mode has been turned off, it cannot be reactivated. This ensures data integrity after the Admin has decided to continue with regular business operations.

  • The Test Mode banner now has a clickable link to access Test Mode and Batch Delete settings.
