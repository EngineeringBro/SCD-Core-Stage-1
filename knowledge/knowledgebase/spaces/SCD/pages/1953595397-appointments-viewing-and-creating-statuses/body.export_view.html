Users can reserve blocks of time for your customers to come in and spend time with an assigned user. You can also specify whether an appointment is for a Repair ticket, Sales ticket, or a Trade-in ticket. If a customer decides they need to change their reservation, users can change the appointment from the Appointment List.

Admins/Managers can set the state attributed to each appointment status, as well as change the order of the statuses, and even add new statuses. A color is assigned, based on the classification of the status.

  • Open - Blue

  • Closed/Committed - Gray

  • Canceled - Red

  • No show - Orange

Viewing and Adding Appointment Statuses

 

1. From the Navigation Bar, go to Settings → Company Settings.

2. On the Settings Overview page, look for the Appointments section. Under this area, click Statuses.

3. You can add a new appointment status by clicking the Addd Button (+) on the upper-right side of the page. Alternatively, you can edit a status by clicking the Edit Button next to one of the existing selections.

4. When creating a new status, a pop-up will appear with fields to define the status. Once you have entered the information, click Create to add the new status.

  • Name - Assign a name to your new status

  • State - Assign a state to classify the appointment status: Open, canceled, closed, committed, or no-show.

  • Order - Assign a number to change the order of statuses on the overview page. A lower number will seat an entry higher on the list.

  • Set as Default Status for State - When this is selected, this status will be the default for the assigned state.

Note: Default statuses (Open, Closed, and Canceled) cannot have their state edited, and they cannot be deleted. Once a created status has been used, it is locked from deletion, to maintain historical records.
