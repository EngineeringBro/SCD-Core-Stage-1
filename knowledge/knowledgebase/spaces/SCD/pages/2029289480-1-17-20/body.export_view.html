Released on 9/1/2021 at 11:30pm CT

Minor Changes and Improvements

  • Users with Admin priviliges can now update their company logos in Login, Navigation Bar, and Qsite Customer Portal with image file names that include spaces.

  • Reworked the Sales Summary by Staff export file to include all of the correct data for Percentage of Sales, Ticket Count, and Average Ticket Total columns.

  • Added the SKU column back to the Inventory Count page.

  • Added support for the PAX A80 payment device for use with the Bluefin integrated payment method.

Bug Fixes

  • Resolved an issue where Internet Explorer would not allow a user to create a new ticket.

  • The Compatibility modal now works on Internet Explorer.

  • Fixed the Edit Views modals on the Customers, Invoices, and RMA Tracking list view pages.

  • Using the restock fee feature no longer causes a negative balance erroneously after a ticket has already been closed.

  • Fixed a bug where the automatic clock out timer does not clock users out at the set time
