With RepairQ 1.19.1, we have introduced Catalog Item Quick Create, which allows users to create new catalog items directly from purchase orders and sales/repair tickets.

There are a few things to remember for Quick Create items:

  • Items are created at the Global level, i.e., all locations will have access to this catalog item.

  • All roles that have access to creating tickets can quick-create catalog items.

  • Admins can choose which Item Types are used for Quick Create.

Users can add additional columns to the Catalog Item List to see when an item was created and which staff member created it. Users can also filter that list to view items based on which process was used to create the item.

Add Creation-Related Columns to the Catalog List

1. On the Navigation Bar, go to Settings → Catalog.

2. On the Catalog Item List screen, click the Options button (⋮) on the right side of the page. From the dropdown list of options, select Edit Columns.

3a. Click the Add Button (+), and the Column Edit window will appear. These categories are the current columns on your catalog page.

3b. In the category box, type created and additional column selections will appear.

3c. Under the Add Columns section, select Created By, Created From, and Created On to add them to the column list.

3d. Once you are finished customizing your category list, click the Update button (checkmark icon) to update your list.

4. After updating the columns, your Catalog Item List will reflect the new categories added.

  • Created On - This shows the date when the Catalog Item was created.

  • Created By - This shows which user created the Catalog Item.

  • Created From - This shows which process was used to create the Catalog Item.

Note: You can save this new list view by following the instructions in our Knowledge Base Article, List Views Overview.

Filter the Catalog List by Created Context

When searching the Catalog, there is a new filter named Created From. This allows you to search your list based on the created method of the Catalog Item.
