Compatibility Lists streamline your repair process by linking catalog items with compatible services, parts, and other Item Types. For instance, you can pair repair devices with compatible parts, link parts with the corresponding service, or associate accessories with retail devices. This process reduces errors by preventing the attachment of incompatible items to a ticket and enhances your staff’s efficiency when managing tickets.

Compatibility plays a role in establishing approved parts and service lists for various service programs. Restricting Compatibility for parts with services ensures a seamless and accurate workflow. Additionally, compatibility links provide suggested ticket items, optimizing sales opportunities. Device Compatibility is used for creating part and service relationships in your Item Grid, offering a visual guide for users throughout the repair process.

  • Efficiency and Error Reduction - Associate Repair Devices with services and parts, enhancing the repair process and preventing errors by eliminating the attachment of incompatible items.

  • Service Program Optimization - Build Compatibility Lists for parts and services to streamline your repairs.

  • Visual Guidance - Compatibility Lists work with the Item Grid, which provides visual cues when selecting devices and items during the repair process.

  • Suggested Sales - Seeing all compatible items in the Item Grid helps suggest additional ticket items to your staff, presenting opportunities for upselling and improving the customer experience.

Establish Device Compatibility

Search your Catalog for the device you want to associate compatible items. Next to the item, click the Options button to view available actions. From the dropdown list, click Compatibility.

A Compatibility pop-up for the item will appear. On the left side are classification tabs for Services, Parts, and Other. Based on the item type, select a category then use the Search Box to locate the item you want to mark as compatible. Once you have finished adding items, you can close the window.

Add Compatible Items in Bulk

If you want to add multiple items to the compatibility list, use the Add button on the right side of the pop-up. Use the search fields to look for items you wish to add. Select which items you want to add to the Compatibility List and click Link Selected Items.

When viewing an item’s Compatibility List, all compatible items are listed under Catalog Item.

Enforce Compatibility with Restrictions

Compatibility restrictions will remove non-compatible items (items not added to the Compatibility List) from Smart Search results and the Item Grid. Compatibility restrictions are enforced based on the device assigned to a Repair Ticket or Parent Item in a bundle. This can reduce errors by removing any extra items that could added to the ticket by mistake.

Note: When viewing the Compatibility List for an item, there are separate restriction toggles for Services, Parts, and Other.

When viewing an item’s Compatibility List, use the Restrict Parts toggle to allow only compatible items in Smart Search results and when navigating the Item Grid.

If Compatibility Restrictions are turned off, the user can use the Compatible Only toggle on the Item Grid navigation window to restrict selections to include compatible items only.

If an incompatible item is added to the ticket, the user will encounter an error when saving the ticket. An Admin may need to update the Compatibility List before the item can be added to the ticket.
