Qpop Images

We fixed some issues where some image filenames would not display on Qpop.

Forms with more than one signature were encountering an issue with duplicated signature lines. This issue has been resolved.

The ticket edit screen will no longer complain about not connecting to Qpop when no Qpop companions have been created yet.
