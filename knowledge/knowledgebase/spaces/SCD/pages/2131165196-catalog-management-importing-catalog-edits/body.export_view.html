RepairQ offers convenient inventory management by allowing users to add and update Catalog Items via an import file. Users can effortlessly create catalog items, registering the items and services in your store. This process is streamlined through import files, supporting .xlsx (Excel), .csv (comma-separated values), and .txt (tab-separated values), with .xlsx recommended for the best experience.

  • Efficient Inventory Management - Easily add, modify, and maintain items in your Catalog.

  • Streamlined Data Entry - Quickly upload items in bulk, reducing errors and saving time.

  • Flexibility - Keep up with the latest devices and services as they become available.

Columns are matched on the header name, so you must have the correct header names (provided in the template) to import the column values successfully. Any unrecognized column headers will be ignored; only the values you wish to add or update must be in the import file. If no matches are found for a record in your import file, RepairQ will create a new catalog item record from the import row.

Edit Existing Catalog Items via Import

Catalog items can also be updated in bulk through the import process. We recommend filtering your Catalog list down to the items you wish to update and then downloading the import file to make changes. If a line in your import file matches an existing record in RepairQ, the item will be updated with the new data added to your spreadsheet.

Matches are made when comparing the following categories, in this order: SKU, Supplier SKU, and Name. For example, if you are trying to update an item’s name, include the item’s matching SKU so an update will be made to the name, instead of creating a new item. If no matches are found for a record in your import file, RepairQ will create a new catalog item record from the import row.

The example below shows the changes that will be made - Updating the names of existing items and adding SKUs to other items.

Download the Catalog Import Template

When viewing the Catalog List, use the available filters to scope the list down to the items you need to update. When viewing the results, click the Options button, then click Import. Under the Import Catalog Items header, click Catalog Import Template to download an Excel file with results based on the applied filters, ready for edits.

Update the Template File

After downloading the template, open the file to edit existing items you want to change, then save the document. Matches to existing catalog items are validated by the following categories in a specific order: SKU, Supplier SKU, and Name.

Note: You will create a new catalog item if you add a row with a unique name in the file without an existing, matching SKU or Supplier SKU.

For example, if you are trying to update an item’s name, include the item’s matching SKU (highlighted in green) so an update will be made to the name, instead of creating a new item.

If you are trying to update the SKU for an item, be sure to have the matching Name (highlighted in green) input for that item.

When you have saved the file and are ready to upload your import, click Choose File and select your file from the file explorer. Once the file is selected, click Import. RepairQ will begin processing your import file.

Import Feedback

After importing the items, you will be directed to the import feedback page. This page will inform you of the success or failure of each row in your import file. If a line in the document fails to upload, you will receive a message outlining why it failed. After a successful import, you can check your Catalog Item List to view the changes.
