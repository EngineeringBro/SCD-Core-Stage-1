The Return Repair Report gives you a list of all your Return Repair tickets where items were replaced as part of the repair process. This report can provide insight into how much your return repairs are costing you in lost sales and additional costs.

Viewing the Return Repair Report

1. On the Navigation Bar, click Reports.

2. On the Reports page, click the Returns & Adjustments dropdown along the top row of report categories. From the list of reports, click Return Repairs Detail.

3. Use the available filters to scope results based on the assigned parameters. Once you have set your filters, click Apply Filters to run the report.

  • Date Range - Select a date range for your scope of results.

  • Ticket Status - Choose to view results based on tickets in Closed status, Open status, or all statuses. By default, the report will run for tickets in Closed status, since ticket information is finalized.

  • Location - Select a specific location or an aggregate report for multiple/all locations. 

  • Item Type - Filter by specific Catalog Item Types.

  • Catalog Item - Search for a specific Catalog Item used on a repair.

4. (Optional) On the upper-right of the screen, you can click To Excel to export the report in an Excel-friendly format. You can click Hide/Show filters to collapse/expand the filters when viewing reports.

Return Repairs Report

The Return Repair Report will show a summary of replacement charges, cost, labor fees, and net loss. Below the summary, you will see a list of return repair items, with links to the corresponding original repair and return repair tickets.

Some columns provide additional information and equations for the totals, which you can view by hovering over the Tooltip.
