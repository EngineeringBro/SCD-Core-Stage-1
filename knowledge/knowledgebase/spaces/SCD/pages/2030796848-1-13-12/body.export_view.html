Released 03/19/2018 CT 12:20pm CT (18:20 UTC)

PAX Device "Format Error" Recovery

We have received reports of PAX Devices presenting a "format error" when processing a transaction. When this error occurs, the payment is processed but it is not saved on the RepairQ ticket. With this update, RepairQ will automatically recover from a format error.

The format error happens outside of RepairQ's control, and with a format error RepairQ does not receive a normal response to process. Now when we receive a format error we use Bluefin's APIs to lookup the transaction, and if it is processed we save it onto the RepairQ ticket without any manual intervention required.
