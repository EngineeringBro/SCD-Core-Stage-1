Processing a return allows you to manage and track returned items efficiently, ensuring accurate inventory and customer satisfaction. Returns can be performed on a ticket multiple times until all items have been returned. Once all items have been returned, the ticket will be locked for editing to maintain data integrity. When viewing a ticket, returned items are listed under the "Returned Items" section of the ticket page, providing a clear overview of the return history.

Return an Item/Service on a Ticket

Look up the ticket for the customer. You can use the search tool below the Navigation Bar to search by ticket number or select the Contacts button to search by customer information. Alternatively, you can filter the Ticket Queue for results by searching tickets in Closed status.

When viewing the ticket, click the Return button on the Actions Bar. On the pop-up screen, you will see a summary of the ticket, including a link to any warranty/disclaimers for the customer. Click Start a Return to initiate the return process. (If you are performing a Return Repair, click here for more information.)

Under the "Returned Items" section of the ticket, select the item being returned by clicking the checkbox next to the item. A new, "Initial Diagnostics" section will expand under the returned item. If the returned item is inventoried, select the inventory status for the item returning to inventory. A note is also required, so we recommend adding a reason for returning the item.

Note: There are different inventory statuses for returned inventory. See our article, Inventory Status, for more information on which status to choose.

By default, the full unit cost will be refunded. If you are refunding only a portion of the item’s price, you can set a new refund price of the item in the "Unit Cost" field.

Note: If the Unit Cost of a refund item is set to a value greater than the original sale price, the refund price will reset to the original sale price.

Once all items being returned have been selected, click the Continue button in the "Returned Items" section.

If the customer wants to purchase additional items with the return credit, you can add the items in the "Ticket Items" section.

Once you have added items for return, the Ticket Totals section on the right side of the screen will show a negative balance, reflecting the amount due back to the customer.

Refund a Transaction

Users can either void a transaction to reverse the payment or add a refund transaction to the ticket.

Note: You must assign a VCT to your terminal to perform a refund on returns since this is tracked in your daily Cash Reconciliation report. The available refund methods will reflect which payment methods were assigned when opening the VCT.

Void a Transaction

Void a transaction by clicking the Red Void Icon next to the original transaction. This will create a reverse transaction (return) using the same Payment Method, and retain the original transaction info.

Add Refund a Transaction

Click the Yellow Refund Icon next to the original transaction. This will allow you to add a refund transaction for the original payment or select a different payout method.

Note: A refund can be performed for integrated card payments without the card being present; however, there is a limit of one refund per integrated card transaction.

Select the desired Payment Type and the refund amount, if you are performing a partial refund. Click Refund in full if you are refunding the full amount originally paid. Add a note (optional) and click Submit Refund.

Note: The available payment types will reflect which payment methods were assigned when opening the VCT.

The "Transactions" section of the ticket will list all payments and refunds. This refund will reference the original transaction line in the "Status" column of the refund.

Note: If you need to make changes, you can cancel the pending refund by clicking the red X icon next to the refund line.

Verify the transaction amount. When the ticket balance is $0.00, you can save the ticket and complete the return. When viewing a ticket, returned items are listed under the "Returned Items" section of the ticket page.
