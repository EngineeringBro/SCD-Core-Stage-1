RepairQ’s import process allows users to batch-add or update records efficiently, enhancing data management and reducing manual entry. Users with Admin, Manager, or Inventory Manager roles can use built-in import templates to manage records like Tickets, Customers, Purchase Orders, and Inventory, with specific required values for each category.

We've recently improved system performance by moving the import process to an asynchronous queue. This change allows large file uploads to run in the background, preventing delays and browser timeouts caused by processing-heavy actions, ensuring a smoother experience. The asynchronous process eliminates database bottlenecks by queuing long-running tasks, and notifying users when these processes are complete.

Benefits of importing spreadsheets include:

  • Efficiency - Quickly add or update large volumes of data, saving time and reducing manual entry. Large files are imported in the background, allowing users to leave their current screen while the import is processed.

  • Accuracy - Ensure consistency and reduce errors by using standardized spreadsheet formats.

  • Alerts - Users are notified via email when file imports are completed.

Import Rules

Column Matching - Columns are matched by header name. We recommend using the header names provided in our templates to ensure a successful import of the column values. Any columns added outside the provided template will be ignored.

Required Values - Required values differ based on the import, such as tickets, customers, purchase orders, etc. Refer to the specific import guidelines for each category to ensure all necessary fields are included.

Import Updates - If a line in the import file matches an existing record in RepairQ, the item will be updated with the data from your spreadsheet. Matches are made in the following order: SKU, Supplier SKU, and Name.

Accepted File Formats - RepairQ accepts Excel (.xlsx), Comma-Separated Values (.csv), and Tab-Separated Values (.txt). We recommend using .xlsx files for the best upload experience.
