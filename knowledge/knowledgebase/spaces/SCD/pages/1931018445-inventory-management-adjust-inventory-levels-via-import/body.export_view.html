Users with the Inventory Manager role or higher can make bulk inventory adjustments by importing data on the Inventory Summary page. This provides an alternative to manual adjustments or inventory counts, allowing users to adjust items into inventory (Instock) or out of inventory (Shrinkage).

This import also allows you to add inventory items that are out of stock (0 quantity) by including them as new rows at on the template. Once the required fields are filled in and a quantity is entered, the import tool will update your inventory accordingly.

Note: These adjustments will only affect the Instock status of items. Items listed in other statuses: Pulled, Ordered, etc. will be omitted from the Import Template. For additional information on understanding Inventory Status, see Inventory Status: What Does it Mean?

Import Rules

  • Column Matching - Columns are matched by header name. We recommend using the header names provided in our templates to ensure a successful import of the column values.

  • Unrecognized Columns - Any columns added outside the provided template will be ignored.

  • Required Values - Required values differ based on the import, such as tickets, customers, purchase orders, etc. Refer to the specific import guidelines for each category to ensure all necessary fields are included.

  • Import Updates - If a line in the import file matches an existing record in RepairQ, the item will be updated with the data from your spreadsheet. Matches are made in the following order: SKU, Supplier SKU, and Name.

  • Accepted File Formats - RepairQ accepts Excel (.xlsx), Comma-Separated Values (.csv), and Tab-Separated Values (.txt). We recommend using .xlsx files for the best upload experience.

Download the Inventory Import Template

On the RepairQ Navigation Bar, go to Inventory → Inventory Summary. On the Inventory Summary page, you can filter results for the batch of inventory items you would like to adjust. We recommend paring down the inventory list before downloading the import template, as the template will include all results from the current list. If the list includes all items in your catalog, the download may take longer than expected.

Note: To focus on all items with an on-hand quantity, use the filter “Inventory Focus - Instock.”

Once your new results have populated, click the Options button near the top-right of the page. From the list of options, click Import.

A new section will appear below the summary list — Import Inventory Stock Changes. Click the Inventory Import Template button to download a spreadsheet of your filtered results. Your browser will download the Inventory Import Template in Excel format. The template file will include all of the catalog item filters in the current view, along with all of the columns needed to import stock changes. Save this file somewhere on your computer that is easily accessible for you to make changes and re-upload.

Update the Import Template 

The import template will be pre-filled with data based on your filter results, including key details like Item Name and Location. You'll only need to update the quantity fields as needed. You only need to include the values for any required field and any values you wish to add or update in the import file.

Columns are matched based on the header names provided in the template, so it's important to use the correct headers for a successful import. Any unrecognized column headers will be ignored. Each inventory adjustment must correspond to an existing catalog item in your inventory. Matches are made in the following order: SKU, Supplier SKU, Name, and UPC.

Verify Instock Quantity

When looking at the spreadsheet, Column F represents the current Instock Qty of the item. If you have existing inventory, this value is already populated. If you are adding a row to the template for an item with no current quantity, this should be zero (0).

Updated the Counted Quantity

Column G shows the Counted Qty, which is for the quantity of inventory that you have physically verified as counted for this item. You will enter the counts for each item here.

Fill in the Adjustment Quantity

Column H is where the Adjustment Quantity will be recorded. This total is used to apply inventory count changes. To assist with this, you can type the formula =G2-F2 into cell H2, and press Enter to update the cell. You can also drag the right corner of the cell down to lower rows to copy the formula to the others.

This formula subtracts the Instock Quantity from the Counted Quantity and leaves you with an Adjustment Quantity.

  • A zero total in Column H means there is no variance between your on-hand counts and your counted quantity. This item will be ignored when imported into RepairQ.

  • A positive number in Column H means there is extra stock, so this is a positive variance.

  • A negative number in Column H means there is a shortage of stock, so this is a negative variance.

Reconcile Variances on the Import Document

Before you upload the document to RepairQ to update your counts, variances require additional information in the document. This will ensure a successful upload.

Adjustments for Positive Variances (Add Inventory)

If the Adjustment Qty is greater than zero, new inventory items are added for the catalog item and location specified. The following columns are required when adding inventory:

  • Adjustment Qty - The amount that you will be adding to inventory.

  • Location - The Short Name for the location you are making the adjustment.

  • Supplier - The Supplier for this catalog item. This must match an existing value in RepairQ.

  • Condition - The condition of this inventory item. This must match an existing value in RepairQ.

  • Serial Number - The Serial Number assigned to the device. Leave this field blank if the item is not serialized.

    • If the adjustment is for a single serialized item, enter the serial number under column I.

      • If the Serial Number is numeric, you may need to add an apostrophe ( ' ) before the number to override the cell formatting. e.g. '12345.

      • If you are adjusting multiple Serial Numbers for the same item, you can enter the Serial Numbers separated by a comma, with no spaces. e.g. '15926140,1592649.

Adjustments for Negative Variances (Inventory Shrinkage)

If the Adjustment Qty is less than zero, inventory items are subtracted for the catalog item and location specified. The following columns are required when shrinking inventory:

  • Adjustment Qty (a negative number equal to your adjustment)

  • Location - The Short Name for the location you are making the adjustment.

  • Serial Number (if the item is serialized)

    • You will need to enter the Serial Number of the item that will be removed from inventory under Column I.

      • If the Serial Number is numeric, you may need to add an apostrophe ( ' ) before the number to override the cell formatting. e.g. '12345.

      • If you are adjusting multiple Serial Numbers out of inventory, you can enter the Serial Numbers separated by a comma, with no spaces. e.g. '12345,12346.

With a successful import, existing in-stock inventory items change to the status specified in your import file. If no status is specified, the items will default to "Shrinkage" status.

Add Rows for Inventory Adjustments (Optional)

Whether you're using an existing template with catalog items or starting with a blank template, you can add rows for multiple items that need adjustments. Each inventory adjustment must correspond to an existing catalog item in your inventory. Matches are made in the following order: SKU, Supplier SKU, Name, and UPC.

If you are adding a new row to the template to adjust inventory levels, you must enter information in the following columns:

Note: Except for quantity adjustments, all other fields must match existing values in RepairQ.

  • Location - The Short Name for the location you are making the adjustment.

  • Item Type - The Item Type assigned to the catalog item. This must match an existing entry in RepairQ.

  • Manufacturer - The manufaturer of the catalog item.

  • Name - The Catalog Item name. This must match an existing entry in your catalog.

  • Instock Qty - The current Instock quantity of the item. If you are adding an item with no current Iinstock quantity, this should be zero (0).

  • Counted Qty - The quantity of inventory that you have physically verified.

  • Adjustment Qty - The sum/difference between your Instock Qty and Counted Qty.

  • Supplier - The supplier/source of the inventory item.

  • Condition - The condition of the inventory item.

  • Cost (Optional) - If undefined, this information will pull what is set for the catalog item.

  • Price (Optional)- If undefined, this information will pull what is set for the catalog item.

  • Status (Optional) - You can set a status other than Instock for new items, however, if the status is not defined, it will default to Instock.

Importing the Template and Import Feedback Page

When you have saved the file and are ready to upload your import, click Choose File and select your file from the file explorer. Once the file is selected, click Import. RepairQ will begin processing your import file in the background, and you can continue working. You will be notified via email when the import is complete, and you can check the Import Queue for any additional details.
