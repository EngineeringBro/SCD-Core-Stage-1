A Category Tree is a structured grouping of categories for your Catalog Items, organized in a parent/child hierarchy. It provides a top-down method for building a visual browsing experience within Item Grids. Together, the Category Tree and Item Grid create an intuitive, drill-down interface for selecting products and services.

These features help structure your offerings in a way that enhances both customer interaction (using the Qsite Opportunities Portal) and staff efficiency when adding products and services to tickets. By implementing a Category Tree, you can create a seamless, user-friendly navigation system that elevates your point-of-sale experience. See a video demo of Item Grids in action here.

Enhance Navigation with Item Grids

After establishing your Category Tree, the next step is customizing the browsing experience by configuring the Item Grid. This functionality enables users to tailor the visual presentation of categorized items built in your Category Tree, incorporating graphics and assigning Catalog Items to respective categories.

Access your Item Grids

Navigate to Company Settings → Catalog → Item Grids. Item Grids are generated based on your existing Category Trees. Select a category to configure its corresponding Item Grid.

Note: You can create a new category for your Category Tree by clicking +Add → Category on the upper-right of the page.

Link Inventory Items to a Category

Catalog items can be linked from the Item Grid page, catalog list, and catalog import.

Add Catalog Items from the Item Grid (Method 1)

First, select a category tree and navigate through its levels to locate the category you wish to link catalog items.

Note: When using the drill-down to navigate the Category Tree, you can see a breadcrumb trail of categories at the top of the page. Each level is listed hierarchically, with the current subcategory highlighted for easy reference. If you need to step back to a higher category, click a category on the path to navigate back up.

Click the Add Button (+) and select Catalog Item. Use the Advanced Search to locate the item(s) you want to add to the category. Check the box next to the catalog items you want to link to a subcategory and click Link Selected Items.

Linked catalog items are displayed below the subcategory highlighted in the trail of categories.

Assign Categories From the Catalog (Method 2)

You can also add items to a category from the Catalog. Go to your catalog and filter for the item(s) you want to assign to a category. Check the box next to the item(s) you wish to assign, then click the Actions button on the top right of the list. From the list of options, select Assign Category.

Next, select a category tree and navigate through its levels to locate the category you wish to link catalog items. Once you are in the category, click Assign.

Note: When using the drill-down to navigate the Category Tree, you can see a breadcrumb trail of categories at the top of the page. Each level is listed hierarchically, with the current subcategory highlighted for easy reference. If you need to step back to a higher category, click a category on the path to navigate back up.

Assign Categories via Catalog Import (Method 3)

Use the Inventory Update import to bulk-assign categories to your catalog items. Start with the Root Category and drill down to the preferred Category. If the item will be linked to multiple trees, they are comma separated.

Note: Category names must match existing categories in your Category Tree structure.

Remove Items from a Category

If there are any items you want to remove from a category, Grid View allows you to select multiple items from the Catalog Items section and then Remove from Category.
