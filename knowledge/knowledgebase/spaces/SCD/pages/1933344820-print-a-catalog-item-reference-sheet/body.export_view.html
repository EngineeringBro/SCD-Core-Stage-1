Users can create customer label pages for price sheets, label printing, and more. Printing a list of items can be helpful to your staff, such as reference sheets for common sale items/services to scan at checkout and inventory scan lists for items that don't have their own labels.

Printing a Catalog Item List Sheet

1. On the Navigation Bar, go to Settings → Location Settings.

2. On the Settings Overview page, click Catalog from the list of options on the left column. From the dropdown, click Print Label Pages.

3. Use the filters to customize your page settings, display information, and catalog items on your list.

  • Page Title - Create a title for the page.

  • Columns/Rows - This controls the number of rows and columns per page. By default, the sheet will print in two columns with four rows on each page.

  • Page Numbers - Select this option to include the page number on the label page.

  • Display Settings - Select what catalog item details will be included on the page.

  • Barcode Format - If opting to print a barcode, select the format. The most common format is Code39.

  • Barcode Size - If printing a barcode, select the height, in pixels.

  • Item Type - Filter results by a specific item type.

  • Manufacturer - Filter results by a specific manufacturer.

  • Catalog Item - Filter for results that include a specific keyword.

  • Filter Focus - Select whether the list will include inventoried items, serialized items, and if the list will include inactive catalog items.

4. After you have made your selections, click Apply Filters. Your list will show as a preview below the filter section.

5. Click the Print button at the top-right of the screen to print the page.
