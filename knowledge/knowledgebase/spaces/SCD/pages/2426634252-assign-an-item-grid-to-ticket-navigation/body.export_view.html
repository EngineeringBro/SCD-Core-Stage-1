A Category Tree is a structured grouping of categories for your Catalog Items, organized in a parent/child hierarchy. It provides a top-down method for building a visual browsing experience within Item Grids. Together, the Category Tree and Item Grid create an intuitive, drill-down interface for selecting products and services.

Item Grids and Ticket Item Look-up

Item Grids are connected to the “Look up” button on tickets and adapt to the context where the button appears. Different grids can be assigned when searching for Repair Devices, Trade-in Devices, and Ticket Items.

For Instance, when working on a Repair ticket, assign the "Repairable Devices" grid to the Repair Devices section of the ticket. This grid may contain items eligible for repair, facilitating the selection of repair items for the ticket. The “Items for Sale” grid can be assigned to sell inventory when adding a sale item to any ticket.

Assign Item Grids to Ticket Item Look-up

Once you have added Catalog Items to your Item Grid, select which grid will be used for navigation. Go to Company Settings → Tickets → General Settings.

Assign an Item Grid to the Item Look-up Button

The Ticket General Settings page has sections for Repair Items, Trade Items, and Ticket Items. Each section has separate Item Grid assignments. After making your changes, be sure to Save your settings.

  • Repair Items - This will assign a grid to the Look Up button in the “Repair Devices” section of Repair Tickets.

  • Trade Items - This will assign a grid to the Look Up button in the “Trade-in Items” section of Trade-in Tickets.

  • Ticket Items - This will assign a grid to the Look Up button in the “Ticket Items” section of all tickets.

  • Use Item Grid to add items - Toggle ON to navigate the Item Grid when using the Look up button to add items to a ticket. When turned OFF, the search will default to Advanced Search.

    • Item Grid - Assign which item grid will be used for looking up items.

    • Preview - Click to preview the grid you have assigned for this Ticket Type.

  • Item Grid Name - Use these settings to customize which catalog attributes will be shown when viewing Grid Items.

    • Name - Show/Hide the catalog-assigned Name on Grid Items.

    • Short Name - Show-Hide the catalog-assigned Short Name on Grid Items.

    • SKU - Show/Hide the catalog-assigned SKU on Grid Items.
