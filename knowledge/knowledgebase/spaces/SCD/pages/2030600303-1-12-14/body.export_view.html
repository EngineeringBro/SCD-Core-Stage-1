Released 07/18/2017 11:52 AM CT (16:52 UTC)

Customer Credit Balance Updates

When processing two tickets simultaneously for the same customer where both tickets have store credit transactions, the customer's balance would not take into account both changes simultaneously, and as a result the customer's credit balance would be incorrect. We have resolved this issue.

Staff Management Permissions

We have tightened up a few areas of staff management so that managers cannot edit certain attributes of admin users.

RMA Batch Location Filter

We added a location filter to the RMA batch list page.

Notification Editor

The email notification content editor now includes an HTML mode so that you can paste in or tweak the raw HTML markup.

PAX Device Integration

We added an additional layer of validation to harden the integration against missing or mis-identified responses.

Login Timestamps on System Log

We added timestamped login info to the System Log. Now when viewing the System log, you can see when staff members login to your RepairQ app, as well as the IP address they accessed it from.
