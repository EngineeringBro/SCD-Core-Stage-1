This update includes a few quick updates.

Enterprise Data Feed

  • Fixed an issue related to the new database schema introduced in 1.7.4 which will restore the data feed functionality.

Dialog / Modal Windows

  • In RepairQ there are many cases where a dialog / modal window overlays the screen, and some windows would be more effective if they covered more vertical space. With this update, window overlays will stretch vertically to use more of the browser window.
