Admins can activate customer Zip Code validation when adding customers in the United States, Canada, Australia, and Great Britain countries. When a user creates or edits a Customer Profile, the system will perform a validation check to make sure the zip code is valid for the assigned country.

Note: This validation is based on the country selected in the Customer Profile, so users are able to add international customers, as long as their address is in the US, CA, AUS, and GB. The system will validate the zip code for the correct country, but will not validate for the specific city/state/region.

Hyphenated codes for CA will not be accepted. The system will recognize the code without symbols.

Activate Zip Code Validation

1. On the Navigation Bar, go to Settings → Company Settings.

2. On the Settings Overview page, look for the Customers category. Under this section, click General Settings.

3. On the General Settings screen, toggle ON the option for Validate zip code for US, CA, Au and GB countries. Be sure to Save your settings.

Note: Zip code validation will override any Validation Rules set for the zip code field. When active, a zip code will always be required for customer profiles.

Zip Code Validation

When validation is active, the user will be notified that a blank/incorrect Zip Code field does not match. You will need to enter a valid zip code before you can proceed.
