Locating the Appointment List

To find your Appointment List, use the Navigation Bar at the top of RepairQ to go to Opportunities → Appointments.

Viewing Set Appointments

After an appointment is set, it will appear on the Appointments List View page. 

The Appointments List view page allows users to search for appointments by Name, Device, Request Type, and more. For more on how list views and searching work, please see our article, Custom List Views, Filters, and Columns.

You can view scheduled appointments by List, Day, Week, or Month. In each view, you will notice a left and right arrow that allow you to cycle between units of time. There is also a today button, which when viewing the Day list, will bring you back to the current date.

Editing an Appointment

We'll walk you through a quick example using Bob Roberts' name as a keyword to narrow results to find their November 21st appointment and change the date to November 28th. 

1. From the Appointments List View filter, select +Keyword.

2. Enter your keyword (Bob Roberts), then select Apply.

3. From the narrowed results, scroll through to find the appointment you would like to edit and select the Options Menu below the Action column.

4. From the Options Menu, select Edit.

5. Review the information shown in the Edit Appointment pop-up, and find what you would like to edit.

6. For this example, we will change the Date and Time. Click each field and edit the information as needed.

7. When you are finished, click Save to submit your changes.

When you are finished, the list will refresh, and you will be able to see the changes to your appointment.
