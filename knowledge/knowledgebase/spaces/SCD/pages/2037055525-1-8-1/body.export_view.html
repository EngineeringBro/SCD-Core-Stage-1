This is a minor release with some fixes and enhancements, primarily to the new global search feature.

Ticket Searches

In an effort to keep search results focused, we were excluding closed tickets from many searches, unless explicitly included in the ticket status filter. We have relaxed this condition considerably. Any time a search keyword is present and no statuses are explicitly filtered, closed tickets will show on the results page.

Search Optimizations

We optimized the ticket search considerably. Particularly where it comes to searching for numbers (phone numbers, ticket numbers, etc), RepairQ would sometimes not return results in an acceptable amount of time.

Customer Search

There were some customer search persistence issues. If you had an active customer search it would override any search keywords used in the global customer search box. This issue has been resolved.
