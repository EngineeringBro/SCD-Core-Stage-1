If you are utilizing Bluefin or Heartland for payment integration in RepairQ, you will need to download and install a Security Certificate on each computer that will communicate with a PAX device.

This tutorial will guide you through the process of downloading, installing, and testing the required certificate.

Downloading the PAX Security Certificate

1. Download the certificate: ca.repairq.io.cer

2. In the Downloads folder on your computer, double-click the downloaded certificate to begin the installation.

Installing the PAX Security Certificate

1. After opening the file, there will be a pop-up for the Certificate Import Wizard. Click Next to begin the process.

2. On the Certificate Store screen, select Automatically select the certificate store based on the type of certificate, then click Next.

Note: Alternatively, we recommend placing the certificate in the following store: Trusted Root Certificate Authorities.

3. Click Install Certificate to begin installing the security certificate.

4. Click Finish to finalize the certificate import.

5. A dialogue box will appear with the result of the certificate import. Click OK to finish the process.

Testing the PAX Security Certificate

1. In Chrome, browse to https://test.term.repairq.io to verify that the browser is configured to trust the security certificate. Click on the lock (highlighted in green) to view the security certificate.

2. Click Connection is Secure, then click Certificate is Valid to view the certificate.
