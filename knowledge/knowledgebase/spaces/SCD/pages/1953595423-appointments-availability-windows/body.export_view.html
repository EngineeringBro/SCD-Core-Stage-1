Users can reserve blocks of time for your customers to come in and spend time with an assigned user. You can also specify whether an appointment is for a Repair ticket, Sales ticket, or a Trade-in ticket. If a customer decides they need to change their reservation, users can change the appointment from the Appointment List.

Availability allows locations to set windows to limit the maximum number of appointments during a scheduled time period. This is useful to limit/expand the number of concurrent appointments based on the staff scheduling, peak hours, and other scenarios where you would benefit from adjusting your appointment traffic. For instance, if you wanted to modify a block of time each morning when you know you are short staffed, you can limit the number of concurrent appointments. The same applies to the opposite – if you are fully staffed during a high-demand time, you can set the availability to allow for more appointments during that set time.

Note: Appointment Availability currently applies to appointments that are set up inside RepairQ; these do not apply to appointments made through Qsite.

Viewing Appointment Availability

Currently, Availability Windows can be customized at the Global level, affecting all locations.

1. From the Navigation Bar, go to Settings → Company Settings.

2. On the Settings Overview page, look for the Appointments section. Under this area, click Availability.

Availability Settings

Availability windows allow appointments to be scheduled during a set time, as defined below. Concurrent appointments will set the maximum number of appointments during that specific window. Click a day of the week to expand the details and add/remove availability windows. Be sure to Save your settings before leaving the page.

  • Use availability windows to limit when an appointment can be scheduled - Toggle this to turn ON/OFF the availability windows that are defined for each day of the week. Turning these settings ON will override the settings assigned at the Location level. These windows will also limit which days you can accept appointments. If there is no window set for a day, and this setting is active, then no appointments can be made on that day.

Selecting a Day of the Week

Each day of the week reflects the store hours set in the Location Information page. Click a day of the week to expand the details and add/remove availability windows. Click Add Window to create a new appointment window.

Note: Availability Windows will also limit which days you can accept appointments. If there is no window set for a day, and this setting is active, then no appointments can be made.

  • Window Name - Assign a name for this appointment window.

  • Start Time - Set a start time for the window.

  • End Time - Set an end time for the window.

  • Appt Duration - Set a duration time for each appointment. The available options are based on your Appointment General Settings.

  • Concurrent Appts - Set the maximum number of appointments during each appointment duration window, i.e., this is the number of appointment windows allowed to happen at the same time. Setting a limit to concurrent appointments restricts scheduling to avoid overlapping appointments. For example, one concurrent appointment allows for one appointment at a time.
