Viewing the Appointment List (Appointment Management Page)

From the Navigation Bar, go to Opportunities → Appointments.

Creating an Appointment

1. When viewing the Appointment Management page, click the Add (+) button to create a new appointment.

2a. A New Appointment pop-up will appear. In the Contact field, begin typing the customer's name. The Smart-Search feature will look for existing customers based on First Name/Last Name/Phone Number. If you see an existing entry for the customer, you can select their entry and add their info to the appointment.

2b. If there is no existing customer, you can add a customer by clicking the Add Button (+) next to the Contact field or by clicking + Contact under the Contact entry. This will allow you to create a new Customer Profile.

3. Once you have added the customer, click on the Date field. You can enter this information manually or you can choose a date from the calendar menu.

4. Next, click the Time field and enter or select the time of the appointment

5. Next, click the Duration field and select an option from the dropdown menu. The available range of durations can be set in Appointment Settings.

6. Click the Location Field and select the location for the appointment. This will default to your current location.

7. (Optional) If you would like to assign the appointment to a specific user, click the Assignee Field. This dropdown menu is populated by staff from the chosen location.

8. (Optional) Click the Request field to select the type of ticket associated with this appointment. When the assignee starts the appointment, this will create a ticket based on the request selected.

Note: If Repair or Tradein was chosen, there will be additional fields to add the device and/or serial number and problem description.

9. (Optional) Add any supporting notes, if any.

10. After you have entered all required and relevant information, select Save. The new appointment will show on the Appointment List, where you can View, Edit, Begin, or Cancel the appointment.
