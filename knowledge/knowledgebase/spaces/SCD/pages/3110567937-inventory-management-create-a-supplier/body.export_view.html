In RepairQ, managing suppliers is essential for smooth procurement and inventory management. Supplier information is necessary when creating a Purchase Order or replenishing inventory. Purchase Orders help track products and parts ordered from external suppliers, replenish stock levels, and track spending. Supplier management also allows you to add RMA contact information and return details, making it easier to handle product returns and streamline the return merchandise authorization process directly within RepairQ.

  • Purchase Orders - Generate accurate POs by linking products directly to their suppliers.

  • Spending Insights - Track supplier-specific costs to better analyze purchasing trends and manage budgets.

  • RMA Process: Manage product returns by storing supplier RMA contacts and return instructions in one place.

View the Supplier List

From the Navigation Bar, go to Settings → Company Settings. Under the Catalog section, select Suppliers.

Note: This list can also be managed at the Location level by going to Location Settings.

The Supplier Overview page shows a list of pre-populated and added suppliers.

Note: You can edit existing suppliers by selecting the Edit Button (pencil icon).

Create a New Supplier

To create a new supplier, select Add Supplier. Fill out the required and optional fields as needed. At minimum, the Supplier Name and Is Active checkbox are required. After adding the supplier details, be sure to Save your changes.

Note: To remove (deactivate) the Supplier from dropdown lists and reports, uncheck the Is Active field.

General Supplier Information

  • Location - Select the location that will use this Supplier. Admins can select All Locations if this supplier is used at all locations.

  • Name - The full legal name of the supplier or vendor.

  • Short Name - An abbreviated version of the supplier’s name, helpful for quick identification in reports and POs.

  • Address 1 & Address 2 - The supplier’s physical address, with Address 2 for additional details like suite or unit numbers.

  • Country, City, State, Zip - Full location details for accurate shipping, POs, and return documentation.

  • Phone - The supplier’s main contact number for inquiries about orders or returns.

  • Email - The general email for the supplier, used for communicating about orders or returns.

  • Primary Contact - The name of the main person responsible for managing your account or orders.

  • Website - A link to the supplier’s website for quick access to support.

  • Note - Any additional details about the supplier.

Procurement & Order Management

  • Express Leadtime - The expected delivery time (in days) for expedited orders, helping plan urgent stock replenishments.

  • Standard Leadtime - The usual delivery time (in days) for regular orders, important for managing inventory levels and timing reorders.

  • Payment Terms - The agreed-upon payment schedule with the supplier (e.g., Net 30), useful for financial tracking and planning.

RMA (Return Merchandise Authorization) Details

  • Return Policy - A summary of the supplier’s return guidelines, outlining what items are eligible and under what conditions.

  • Return Method - The preferred method for returning items.

    • Email - Indicates that RMAs should be submitted by emailing the supplier.

    • RMA Through Website - Indicates if RMAs can be submitted through the supplier’s website.

  • Return Period (Days) - The number of days you have to return items after receiving them, based on the supplier’s policy

  • Return Period Start Date - Specifies whether the return window starts from the shipment date, delivery date, or invoice date.

    • Ordered - The return period starts from the date the Purchase Order was placed with the supplier.

    • Received - The return period starts from the date the items are physically received at your location.

  • RMA Email - The contact email for initiating the RMA process through the supplier.
