1.17.13 is a minor update including some small features and bug fixes.

11:30 CT on September 28, 2020

Bug Fixes and Improvements

  • Invoice Creation

    • Fixed display of device serial and description - During the first step of creating an invoice, the columns for device description and serial number were blank. We fixed this so that now you will see the device description and serial number. This will help you confirm that you are creating invoices for the correct device(s).

    • Fixed an issue with filter on invoice creation - When filtering invoices through the end of a month, invoices for the last day of the month were being excluded. This has been fixed so that invoices from the last day of the month are included.

  • Integrated Payments

    • Fixed a duplicate record error - In some cases, a record of a credit card transaction was duplicated, leading users to believe a credit card might have been charged (or refunded) multiple times for the same transaction. We added additional measures to prevent duplicate records from being created for a single transaction.

  • Refurb tickets 

    • Refurb tickets were getting stuck in an open status in some cases. We fixed the issue so that you can no longer stock the refurb item into inventory without also closing the refurb ticket.

  • App Performance

    • We reduced the time it takes to load a number of pages throughout the app.
