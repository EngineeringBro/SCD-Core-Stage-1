Release 06/05/2018 2:15PM CT (19:15 UTC)

Qsite - Get Directions Link

We added a "Get directions" link after booking an appointment with Qsite. This will open Google Maps to provide directions to the store address entered in RepairQ.

Qsite Error Feedback

When a user is walking through Qsite for a quote or appointment, and comes across an error, we added the error information, including reference number, to the displayed content on screen. This way, the user will know what went wrong.

Qsite Browser Back Button Support

When a user is walking through Qsite for a quote or appointment, and click the "Back" button in the browser, the global breadcrumbs have been fixed to show the navigation path correctly.

Qsite Change Appointment Day

We fixed an issue that was causing a delay when selecting an alternate appointment day in Qsite.

Qsite Loading Indicator

We added a loading indicator when submitting the form for a quote in Qsite.

Qsite Category Images

We fixed an issue that was causing category images to not be displayed properly on Qsite. The issue was related to resizing, and has been resolved.
