The Manufacturer field is a classification of items in your store to help organize your catalog. This field serves to identify and specify the company or entity that produced or created the item. Manufacturer details are required when creating Catalog Items and are used when filtering lists and reports. By default, we provide a list of popular manufacturers, and you can add more as needed.

This information is essential for several purposes, including:

  • Product Identification - Uniquely identify each item by its manufacturer, ensuring that customers and staff can easily recognize the brand or source of the item.

  • Warranty and Support - The manufacturer’s name is crucial for warranty and support purposes. If a customer encounters issues with a device, knowing the manufacturer is important for seeing warranty replacement and support.

Adding a New Manufacturer

1. On the Navigation Bar, go to Settings → Location Settings.

Note: Admins can add a manufacturer for all locations by going to Settings → Company Settings.

2. On the Settings Overview page, scroll to the Catalog section. From the available categories, click Manufacturers.

3. The Manufacturer Overview page lists all available manufacturers with options to add, edit, and delete (if the manufacturer has never been assigned to an item). On the upper-right of the list, click Add Manufacturer.

4. Begin filling out the fields for the new Manufacturer. When you are finished, be sure to Save your settings.

  • Name - The name of the Manufacturer.

  • Location - Select the location this Manufacturer will be added to. By default, you can only select locations where you have an assigned role, however, Admins can select a location or All Locations.

  • URL - (Optional) Add the Manufacturer's website for additional reference and support.

  • Note - Add any additional reference notes for this Manufacturer.

  • Is Active - Select this box to make the Manufacturer visible on list dropdowns and filters. When unchecked, the manufacturer is Inactive and cannot be assigned to items, and is removed from filters.

5. After saving the new manufacturer, it will be listed alphabetically. The filter at the top of the list allows you to show All Manufacturers, Active Manufacturers, and Inactive Manufacturers. Under the Actions Column, you can edit existing lines or delete an item if it hasn’t been assigned to a Catalog Item.
