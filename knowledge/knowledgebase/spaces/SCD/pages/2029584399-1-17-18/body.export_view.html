Released 4/26/2021 at 11:30pm CT

Minor Changes

  • Removed WarrantyLife, Cellhelmet, LCDCycle, and Pervacio from the Integration Settings pages. These suppliers have also been removed from the Supplier dropdown menu on purchase orders.

  • Added an option to deactivate unused Warranty Claim Providers from the choice list on tickets. Also removed the option from deleting Warranty Claim Providers that had been used in the past. This option to delete showed up in the past, but did not work properly.

  • Improved the UI and print format quality on the Print Label Pages function within Company Settings. The printed pages now adhere to a stricter format for ease of reading and use with barcode scanners.

Bug Fixes

  • Editing information on Sent RMAs now properly saves changes.

  • Resolved an issue where anything following a quotation mark within a repair device's Description field would be erased.

  • Automated SMS notifications that are set to send to {ticket_assignee} now properly generate a notification.

  • Clicking "Customers" on a Billing Agents View page now properly displays customers with said billing agent set as the default.

  • Entered information no longer disappears from the Loyalty Card field when saving a customer profile.
