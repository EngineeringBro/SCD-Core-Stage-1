New Features

  • Inventory Update Reasons help managers track inventory by requiring staff to record the reason the item has been placed in defective status.

  • Part Core Recovery enables stores to recover and track parts that can be sent back to the manufacturer for reimbursement or credit.

  • Appointment settings have been streamlined and Availability settings help manage your appointments during peak times.

Bug Fixes and Improvements

  • Fixed a conflict with the Location Admin and Location Advisor roles.

  • Fixed an issue with Brightstar Program item price.

  • Fixed an issue when using the Quick Create feature on a Purchase Order.

  • Fixed an issue with Daily Report showing Trade-ins as purchases and sales.

  • Fixed an issue affecting the "Set up your catalog" link in the Inventory Summary.

  • Fixed an issue affecting Billing Agent list visibility.

  • Shortened the Qsite URL link when using shortcodes for SMS notifications and fixed the tracking link directing to Qsite.
