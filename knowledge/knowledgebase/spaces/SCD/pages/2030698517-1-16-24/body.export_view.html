Deployed 3/2/2020 11:30PM CT (16:30 UTC)

Release 1.16.24 is a minor release which adds a handful of new features and bug fixes.

New Features and Enhancements

  • Invoice Created On Date can now be printed on invoices through a new print setting.

  • RepairQ will now prevent changing the item type for already used items via catalog item import.

  • Auto clock out will now set the clock out time with more precision.

  • RepairQ will now auto-close appointments when a quote is converted, and convert a quote when an appointment is started.

Bug Fixes

  • We fixed a handful of issues with capturing electronic signatures on Qpop.

  • Fixed confusing input focus change on serial number input for Trade-in items.

  • Fixed issue with searching some special characters.

  • Fixed error when geocoding locations in Qsite settings.

  • Fixed Summary Report layout issue when viewed from beginning of year.

  • Fixed the ability to clear customer info by saving empty values for customer attributes.

  • Fixed an issue where timesheet totals were incorrect if a staff member clocked in at multiple locations for the same time period.
