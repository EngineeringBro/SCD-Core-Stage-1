This update includes a few minor features and bug fixes.

Ticket Import

The location imported for each ticket in a ticket import will now be set as the ticket location, rather than the location you are logged into when performing the import.

Repair Device Report

The following fields were added to the Repair Device Report:

  • Ticket Assignee

  • Warranty Provider

  • Customer ID

  • Billing Agent ID

  • Billing Agent Company

  • Billing Agent Contact

Inventory Import Template

The Instock qty will now be filtered by the location the user is logged into.

Catalog Item Import

To help prevent import mistakes, an error will now be thrown when a catalog item is imported as serialized, but not inventoried.
